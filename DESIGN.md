# Kiwami Capture Design

## Product Tone
- Calm, premium, craft/manufacturing, local-first AI
- Presentation-ready without flashy SaaS styling

## Colors
- Warm white background
- Charcoal text
- Muted bronze/gold accents

## UI Structure
- Master Demonstration
- Key Moments
- Practice Memory
- Expert Review

## Primary Phrase
- Expert Motion → Practice Memory

## Primary CTA
- Start Capture with Liquid LFM

## Avoid
- Cyberpunk styling
- Generic Bootstrap layouts
- Fake analytics
- Score dashboards
- Childish visual language

## Use
- Large readable cards
- Clear hierarchy
- Restrained accents
- Clean spacing
- Calm, serious product language

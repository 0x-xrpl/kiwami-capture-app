Local-first Skill Capture

Kiwami Capture is a local-first skill capture product built for private workplace and craft clips where the raw motion should stay local, but the learning essence should become reusable.

YouTube shows the demonstration. Kiwami Capture preserves what successors should notice and practice.

The product is not trying to be a full video manual tool or a general SOP engine. It focuses on master-only capture or optional comparison capture, then turns the result into Practice Memory.

Why local / edge AI matters

Workplace clips may contain private techniques, safety-sensitive operations, tools, faces, or customer/site details. Kiwami Capture keeps raw clips local and shares only the distilled learning card.

This matches the product goal: capture what successors fail to see without turning raw internal video into a public artifact.

Optimization Loop

Kiwami Capture uses Liquid LFM in the text Practice Memory generation layer. Raw videos stay local, Practice Memory stays editable, and approved expert-reviewed sessions can become training JSONL records for skill-specific Liquid model optimization.

Current runtime uses llama.cpp / GGUF. The architecture is designed for mobile / edge deployment paths such as LEAP, and the AMD / ROCm path can accelerate local inference where available.

Visual Layer: Liquid Vision selected keyframe analysis uses 1-2 master keyframes when the local vision server is available. Audio / Spoken Hint is an optional context layer for short tool explanations, spoken hints, work sounds, or rhythm cues.

Liquid LFM text layer

Liquid LFM is used in the text Practice Memory generation layer. It generates Practice Memory from structured observations and the master hint through a local model server.

Visual Layer: {{ data.visual_layer or 'Local Keyframes' }}. When Liquid Vision is available, the app uses selected master keyframes and passes the visual observations into the text layer.

Why this is different

Kiwami Capture is not a video manual editor. It is not a general video summarizer. It is not only a SOP generator.

Tebiki creates video manuals. Video-to-SOP tools create procedures. Kiwami Capture captures what successors fail to see.
